# 우리 반 아침모임

유치원 아침모임용 Next.js 웹앱입니다.

압축을 푼 뒤 폴더 안의 파일 전체를 새 GitHub 저장소에 업로드하세요. Vercel에서 저장소를 Import한 뒤 Environment Variables에 `GEMINI_API_KEY`와 선택 항목인 `GEMINI_MODEL=gemini-3.1-flash-lite`를 추가하고 Deploy합니다. API 키는 GitHub 파일에 직접 입력하지 마세요.

Gemini 연결에 성공한 경우에만 화면에 `Gemini AI 생성`이라고 표시됩니다. 키가 없거나 호출이 실패하면 `예시 문장 · API 연결 확인`이라는 안내와 함께 예시 문장이 표시됩니다.
